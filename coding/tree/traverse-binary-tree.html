# 遍历二叉树 traverse binary tree

# Description

满二叉树

完全二叉树

# 特征

  • 在二叉树中，第 i层上至多有2i−1个节点（i≥1)
  • 深度为k的二叉树至多有2k−1个节点（k≥1）
  • 对一棵二叉树，如果叶子节点的个数为n0，度为2的节点个数为n2，则n0=n2+1
  • 具有n个节点的完全二叉树的深度为⌊log2n⌋+1
